библиотеку «container/heap», но это не является обязательным). 

    Подарки сравниваются сначала по Value (самый Value подарок оказывается на вершине кучи). 
    В случае если два подарка имеют одинаковую Value, меньший считается «круче» 
    другого (выигрывает в сравнении).

    type Present struct {
    Value int
    Size int
}
